Central authentication and authorization for a web service.

Running the Demo
================

MacOS X and Linux
::

    export PYTHONPATH=src
    export FLASK_APP=classifier.py
    export FLASK_ENV=development
    flask run

Windows
::

    set PYTHONPATH=src
    set FLASK_APP=classifier.py
    set FLASK_ENV=development
    flask run
